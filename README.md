# LA Accounting – Freelance Bookkeeping & Advisory

This is a simple, static marketing site for a freelance accounting and bookkeeping practice based in Los Angeles.

## Tech stack

- HTML5
- Modern CSS (responsive layout, hero banner with LA skyline background)
- No build tools or frameworks

## Pages

- `index.html` – Home, services overview, CTA
- `about.html` – Background, experience, and approach
- `services.html` – Detailed service breakdown
- `contact.html` – Contact form and direct email

## Local development

Open `index.html` in your browser (or use a simple static file server) to preview the site.

## Deployment

The site is deployed as static files (e.g., GitHub Pages or any static hosting). Pushes to the main branch trigger updates to the live site.
